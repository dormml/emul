
-- The site is written in wiki markup for Jekyll.  See:

This uses Jekyll.  Have Jekyll installed on your system (needs Ruby Gems, v1.8.1 kit, using 'gem install').
(On Ubuntu, just 'apt-get install jekyll'.)

https://github.com/mojombo/jekyll/wiki/Usage

http://maruku.rubyforge.org/maruku.html

http://daringfireball.net/projects/markdown/syntax

-- To re-render the site:

Enter the top-level "emul-web" directory, then run:

jekyll build

Look at the site under /_site (which is not checked in; github does the same dance on push).
